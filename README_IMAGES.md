# Image Placement Guide

Your website is now configured to load images from the `public/images` directory.
Please place your photos in the corresponding folders as described below.

**IMPORTANT: All images must be in .png format.**

## 1. Main Website Assets

| Description | Location | Required Filename | Recommended Size |
|---|---|---|---|
| **Website Logo** | `public/images/` | `LOGO.png` | **Any** (High Res preferred) |
| **Home Banner** | `public/images/` | `home-banner.png` | **1920 x 1080 px** (Landscape) |
